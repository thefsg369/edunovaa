Información del Proyecto EduNova

EduNova es un proyecto desarrollado por Luis Soza, conocido en YouTube como sluisr y también utiliza el nombre FSG en diversas plataformas. Este proyecto fue creado como parte de un trabajo escolar, buscando brindar una experiencia educativa interactiva.

El juego ha sido programado en Python utilizando la librería Pygame y está destinado principalmente para PC. Actualmente, no está disponible para Android debido a la falta de tiempo, pero se contempla esta opción en el futuro.

El desarrollo se llevó a cabo en un sistema Linux, aprovechando las ventajas que este entorno ofrece para la programación y el desarrollo de software. ¡Estamos emocionados de compartir este proyecto con todos y esperamos que sea de utilidad y diversión para muchos!
